# Simulator

Requires OpenFL: <http://www.openfl.org/download/>

Compile and run with `lime test <target>`, where target is one of:

* `flash` (just don't)

* `html5` (does not work anymore since we need `sys.io.File`)

* `neko`

* `cpp`

Or cross compile to:

* `windows`

* `linux`

* `mac`

* `android`

Pass `-args <save>` to load a network save file.
